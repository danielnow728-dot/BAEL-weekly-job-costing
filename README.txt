=========================================
BAEL Job Costing Web Application
=========================================

Hi Team! 

This folder contains the BAEL Job Costing application. To use it:

1. Extract this entire folder somewhere on your computer (like your Desktop).
2. Open the extracted folder and double-click the file named: "run_local.bat"
3. A black terminal window will open. The first time you run this, it might take a minute or two to automatically download the necessary tools.
4. Once it says "Application startup complete", open your web browser (Chrome, Edge, etc.) and go to:
   http://127.0.0.1:8000
5. Drag and drop your Excel file onto the webpage to process it.

IMPORTANT NOTES:
* You MUST leave the black terminal window open while you are using the app. When you are done, you can just close the black window.
* You need to have Python installed on your computer for this to work. If you double click run_local.bat and it says "Python is not installed", please go to https://www.python.org/downloads/ to install it (make sure to check the box "Add Python to PATH" during installation).
